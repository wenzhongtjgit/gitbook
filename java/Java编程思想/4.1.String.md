## String对象是不可变
* String对象真的完全不可变吗？否。可以使用反射，改变字符串对象的value字段，在这个过程中引用指向的字符串对象并没有改变''
## String为什么不可变
* 不仅仅是String，Integer、Long等基本类型也是final的，这就保证了基本类型作为程序的基石，不会被继承。处理这些类型的时候，不需要考虑继承的问题，简化了程序语言的设计；
* 安全，很多类的涉及基于String的不可变，比如Set、Hash等；
* 字符串常量化，String的不可变可以节省很多的内存；
* ....
